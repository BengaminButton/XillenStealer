XillenStealer v3.0 - Инструкция

1. Установка:
- Установите шрифт minecraft.ttf (правой кнопкой → "Установить")
- Установите Python 3.9+ с официального сайта
- Запустите install_deps.bat (Windows) или install_deps.sh (Linux)

2. Запуск билдера:
- Перейдите в папку xillenstealer/dist
- Запустите XillenStealer_Builder.exe
- Пароль: layscrab

3. Создание стиллера:
- Укажите имя (например: WindowsUpdate)
- Получите токен бота у @BotFather
- Узнайте свой ID через @userinfobot
- Выберите нужные функции

4. Что собирает стиллер:
✓ Системная информация (CPU, RAM, GPU)
✓ IP и MAC адреса
✓ Пароли и куки браузеров
✓ Скриншот экрана
✓ Telegram сессии (tdata)

5. Важно:
- Антивирусы могут блокировать стиллер
- Требуются права администратора
- Только для образовательных целей!
