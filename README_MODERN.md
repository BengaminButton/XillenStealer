# XillenStealer Builder - Современная версия

## 🎨 Два варианта интерфейса

### 1. **builder.py** - Основная версия
- ✅ Кастомное окно без стандартных элементов Windows
- ✅ Единый интерфейс без всплывающих окон
- ✅ Профессиональный минималистичный дизайн
- ✅ Исправленная система шрифтов
- ✅ Поддержка Minecraft шрифта

### 2. **builder_modern.py** - Современная версия
- ✅ Использует ttkbootstrap для современного вида
- ✅ Красивые темы и стили
- ✅ Лучшие виджеты и анимации
- ✅ Fallback на обычный Tkinter если ttkbootstrap недоступен

## 🚀 Быстрый старт

### Для основной версии:
```bash
python builder.py
```

### Для современной версии:
```bash
# Установите зависимости
pip install ttkbootstrap

# Или запустите bat файл
install_modern_deps.bat

# Запустите приложение
python builder_modern.py
```

## 🐛 Решение проблем

### Шрифт не загружается:
1. Убедитесь, что `Minecraft.ttf` находится в папке с программой
2. Проверьте права доступа к файлу
3. Запустите `python font_loader.py` для диагностики

### Ошибки ttkbootstrap:
1. Установите: `pip install ttkbootstrap`
2. Или используйте `builder.py` без ttkbootstrap

### Проблемы с окном:
1. Убедитесь, что используете обновленную версию
2. Проверьте, что нет конфликтов с другими приложениями
